📚 Python Library Management System

A Python-based interactive web application developed using Streamlit to efficiently manage library records. The application allows users to add, search, update, delete, issue, and return books while maintaining their availability through an intuitive web interface.

---

📌 Project Overview

The Python Library Management System simplifies library record management by allowing users to maintain book information through a user-friendly interface. Users can add, search, update, delete, issue, and return books while monitoring their availability in real time.

This project was developed as part of my Python learning journey while pursuing the TEVTA Punjab Data Analytics Course.

---

✨ Features

- Add new books
- Display all books
- Search books by Book ID
- Update book details
- Delete books
- Issue books
- Return books
- Display library statistics
- Interactive web interface using Streamlit
